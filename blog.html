Open Graph

Open Graph protocol was developed by Facebook. It lets the users website be compatible with the social media platform the website is being shared with. It does it by the use of meta tags. With meta tags you decide what should be shown while being shared in the social media platform.

Humans.txt

Humans.txt file is a text file which contains information for the humans to read. It has no effect to the contents of the website. It is optional to put it. It may contain details about the author of the website or any other people who deserve credit for the creation of this website.

Robots.txt

Robots.txt file is a text file which contains information for the bots and search engines to read. It has a few lines of codes which specifies which bots this text should concern. It tells the concerning bots which directories (files) are off limits.

Seperation

It is important to separate layout and design (css) from the html because it will make the html file less congested. Also if you were to put the css rules in another css file, those rules could be applied to multiple html files all at once instead of configuring css rules one by one to each html file.
